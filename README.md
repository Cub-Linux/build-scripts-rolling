## Build Scripts

To build an ISO Image execute:
```
sudo ./build.sh -v" in the live-iso folder.
```
  
You can change the OS Version in these files:  
 - live-iso/airootfs/etc/os-release  
 - live-iso/airootfs/etc/xdg/kcm-about-distrorc  
 - live-iso/build.sh
